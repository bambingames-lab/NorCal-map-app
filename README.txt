NorCal Territory Tracker

Files:
- index.html
- manifest.json
- sw.js
- icon.svg

How to deploy on GitHub Pages:
1. Upload all files in this folder to the root of your GitHub repository.
2. Commit the changes.
3. Open your GitHub Pages URL.
4. Use the app normally. It will load real Northern California data from live public sources.

Notes:
- ZIP boundaries come from California's public ZIP boundary dataset and are not official USPS delivery polygons.
- Cities include incorporated cities and CDPs.
